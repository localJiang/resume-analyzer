## 任务状态定义

| 状态 | 含义 |
|-------|---------|
| **待办** | 尚未开始 |
| **进行中** | 正在处理中 |
| **审查中** | 已完成，等待代码审查 |
| **已完成** | 已审查并合并 |

## 优先级定义

| 优先级 | 含义 |
|----------|---------|
| **P0** | MVP 上线必备 |
| **P1** | 高价值，上线后尽快交付 |
| **P2** | 重要但不阻塞 |
| **P3** | 锦上添花，未来考虑 |

---

## 如何添加任务

当 PM Agent 创建新任务时：

```markdown
| # | 任务 | 优先级 | 状态 | 负责人 | 依赖 |
|---|------|----------|--------|----------|--------------|
| X.Y | [任务描述] | P0 | 待办 | [Agent] | [依赖项 #] |
```

随着工作推进更新状态。AI Agent 应在开始前将任务标记为"进行中"，完成后标记为"审查中"或"已完成"。

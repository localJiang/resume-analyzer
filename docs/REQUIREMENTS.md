## 约束条件

1. **技术栈：** Next.js + Next.js API 路由 + PostgreSQL
2. **语言：** TypeScript
3. **认证提供商：** Supabase Auth
4. **部署方式：** Vercel
5. **浏览器支持：** 现代浏览器（最新 2 个版本）
6. **移动端支持：** iOS 15+、Android 12+

---

## 假设条件

1. 用户具备基本的简历文件（PDF/DOCX/TXT），且文件内容为文本（非图片扫描件）
2. LLM API（OpenAI / Claude）在开发期间持续可用，不会因服务中断导致原定功能不可交付
3. MVP 阶段用户数量 < 10,000，不考虑大规模并发和分库分表
